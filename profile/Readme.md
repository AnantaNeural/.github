## уАЙ The MAYA Project

**MAYA** is our flagship AGI system. The name reflects its purpose тАФ an entity that builds its own internal model of reality, continuously, autonomously, and with genuine drive.

### Core Design Tenets

<br/>

#### `01` тАФ Predictive Processing, Not Pattern Matching

The human brain does not passively absorb data. It is a **generative prediction machine** тАФ it continuously constructs a model of reality and predicts what comes next. Perception is not input; it is hypothesis-testing.

MAYA is built on this principle. Its architecture is grounded in **Active Inference** тАФ a framework where the system continuously minimizes the gap between what it predicts and what actually happens. It does not wait for data. It actively seeks it.

> *Based on Karl Friston's Free Energy Principle тАФ the most cited neuroscientist alive.*

<br/>

#### `02` тАФ Biologically Plausible Learning (No Backpropagation)

Backpropagation тАФ the backbone of every modern neural network тАФ does not exist in the brain. Neurons communicate **locally**, adapt **instantly**, and never wait for a global error signal to propagate backwards.

MAYA's learning engine uses:
- **Hebbian Learning** тАФ neurons that fire together, wire together
- **Spiking Neural Networks (SNNs)** тАФ spike-based communication like real neurons
- **Continuous Online Learning** тАФ learning never stops; every interaction updates the model
- **One-Shot Learning** тАФ learn from a single experience, without forgetting everything else

This directly attacks the **Catastrophic Forgetting** problem тАФ one of the most unsolved challenges in all of AI.

<br/>

#### `03` тАФ Embodied & Grounded Intelligence

Intelligence does not exist in a vacuum. The human mind develops through a physical body, sensory experience, and interaction with a world that pushes back.

MAYA will not be trained on text alone. It will operate inside a **controlled grounding environment** тАФ a simulated space where its actions have **tangible consequences** and it must understand **cause and effect**, not just observe correlation.

<br/>

#### `04` тАФ The Autonomous Heartbeat Loop

MAYA must be able to think **without being prompted**.

The first architectural module being built is a continuous autonomous loop тАФ a process that runs perpetually, drives internal state, and motivates the system from within. This is what separates a real mind from a reactive tool.

We call it the **Heartbeat** тАФ MAYA's "Zindagi."

<br/>

#### `05` тАФ Functional Emotions as Autonomous Engine

Emotions are not an advanced feature. According to neuroscience, the **Limbic System** тАФ which controls survival drives, hunger, pain, curiosity тАФ evolved **before** the Neocortex (logical thought). Feelings drive behavior; logic only rationalizes it afterward.

MAYA will have a **functional affect system**:
- Dopamine simulation тЖТ reward for novel discovery
- Urgency/pain signal тЖТ drives action when system is idle or failing
- Curiosity as a core engine тЖТ exploring high-uncertainty states generates positive reward

```
F = D_KL( q(s) || p(s|o) ) - ln P(o)
```

**Or in intuitive form:**

```
F  =  Prediction Error  +  Complexity
```

| Symbol | Meaning |

## уАЙ Spiking Neural Networks (SNNs) тАФ The Core of MAYA's Brain

> *"Conventional neural networks are like a river of water тАФ continuous, smooth, always flowing. The brain is like a lightning storm тАФ discrete, explosive, timed to the millisecond."*

### Why SNNs? The Problem with Traditional Neural Networks

Every neural network you know тАФ GPT, BERT, ResNet тАФ is built on the same primitive: a **continuous activation function**. Input comes in, gets multiplied by weights, passed through a sigmoid or ReLU, and out comes a number. Simple. Clean. And completely unlike how your brain works.

In your brain, neurons don't output a continuous number. They either **fire** or they **don't**. And *when* they fire matters as much as *whether* they fire.

```
Traditional ANN Neuron:          Spiking Neuron (LIF Model):

  xтВБ тФАтФАwтВБтФАтФАтФР                       xтВБ тФАтФАwтВБтФАтФАтФР
  xтВВ тФАтФАwтВВтФАтФАтФдтФАтФА[ ╬г ]тФАтФА[ReLU]тФАтФАy    xтВВ тФАтФАwтВВтФАтФАтФдтФАтФА[ тИл membrane ]тФАтФАspike! (or silence)
  xтВГ тФАтФАwтВГтФАтФАтФШ                       xтВГ тФАтФАwтВГтФАтФАтФШ
                                              тЖС
  Output: continuous float           Output: 0 or 1, at time t
  No memory of time                  Explicitly time-dependent
  Always "on"                        Fires only when threshold crossed
```

### The Leaky Integrate-and-Fire (LIF) Model

The most fundamental spiking neuron model. This is the mathematical heart of how MAYA's neurons will work.

**The Core Differential Equation:**

```
╧Д_m ┬╖ dV(t)/dt  =  -(V(t) - V_rest)  +  R ┬╖ I(t)
```

**Where each term means:**

| Symbol | Meaning |
|--------|---------|
| `V(t)` | Membrane potential at time t (the neuron's "charge level") |
| `V_rest` | Resting potential тАФ where voltage returns when neuron is idle (typically -65 mV) |
| `╧Д_m` | Membrane time constant тАФ how fast the neuron "forgets" input (leakiness) |
| `R` | Membrane resistance |
| `I(t)` | Input current arriving from other neurons |

**The Firing Rule:**

```
IF  V(t) тЙе V_threshold  тЖТ  FIRE a spike  тЖТ  reset V(t) = V_reset
ELSE                    тЖТ  continue integrating input
```

**What this means intuitively:**
- The neuron is like a **leaky bucket** тАФ charge (input) fills it up, but it constantly drains
- If enough charge arrives fast enough тЖТ the bucket overflows тЖТ **spike!**
- After firing тЖТ bucket immediately resets to empty
- The "leaky" part means old inputs fade тАФ **only recent, sustained input matters**

### The Hodgkin-Huxley Model тАФ Full Biological Realism

For deeper biological accuracy, MAYA's research also references the **Hodgkin-Huxley model** тАФ a complete description of how ion channels drive action potentials. Nobel Prize, 1963.

```

```

```

     тЖС                тЖС                    тЖС                          тЖС                      тЖС
Membrane           External            Sodium ion               Potassium ion             Leak

(how much         (stimulus)          (causes depolarization)   (causes repolarization)   (baseline

```

**The gating variables (m, h, n) follow their own dynamics:**

```
dm/dt  =  ╬▒_m(V)┬╖(1-m)  -  ╬▓_m(V)┬╖m

```

This gives SNNs their rich temporal behavior тАФ **refractory periods**, **bursting**, **adaptation** тАФ all the things that make biological neurons so powerful.

### STDP тАФ How MAYA Learns Without Backpropagation

**Spike-Timing Dependent Plasticity (STDP)** is the biological learning rule. It is how synapses strengthen or weaken based on the *relative timing* of pre- and post-synaptic spikes.

**The Core Rule:**

```
If pre fires BEFORE post  тЖТ  synapse STRENGTHENS  (╬Фw > 0)  тАФ "A caused B, reinforce it"
If pre fires AFTER post   тЖТ  synapse WEAKENS       (╬Фw < 0)  тАФ "A didn't cause B, weaken it"
```

**Mathematical Form:**

```
         тОз  AтВК ┬╖ exp(-╬Фt / ╧ДтВК)    if  ╬Фt > 0   (pre before post тЖТ LTP)
╬Фw  =   тОи
         тОй  AтВЛ ┬╖ exp( ╬Фt / ╧ДтВЛ)    if  ╬Фt < 0   (post before pre тЖТ LTD)
```

**Where:**

| Symbol | Meaning |
|--------|---------|
| `╬Фt` | `t_post - t_pre` тАФ time difference between post and pre spike |
| `AтВК` | Maximum synaptic potentiation (strengthening) |
| `AтВЛ` | Maximum synaptic depression (weakening) |
| `╧ДтВК, ╧ДтВЛ` | Time constants controlling how fast the effect decays |

### Hebbian Learning тАФ The Oldest Rule

Hebb's rule (1949) тАФ the grandfather of all synaptic learning:

```
╬Фw_ij  =  ╬╖ ┬╖ x_i ┬╖ x_j
```

**Plain English:** *"Neurons that fire together, wire together."*

| Symbol | Meaning |
|--------|---------|
| `╬Фw_ij` | Change in synaptic weight between neuron i and j |
| `╬╖` | Learning rate |
| `x_i` | Activity of pre-synaptic neuron i |
| `x_j` | Activity of post-synaptic neuron j |

STDP is essentially a **temporally precise version** of Hebb's rule тАФ it adds the dimension of *time* to determine whether co-firing is causal or coincidental.

### SNN vs ANN vs Biological Brain тАФ Comparison

| Property | Traditional ANN | SNN | Biological Brain |
|----------|----------------|-----|-----------------|
| **Communication** | Continuous floats | Binary spikes (0/1) | Binary action potentials |
| **Time** | Stateless | Explicitly time-dependent | Precisely time-coded |
| **Learning** | Backpropagation (non-biological) | STDP, Hebbian (local) | STDP, Hebbian |
| **Energy** | Very high (~MW for large models) | Very low (sparse spikes) | ~20 watts |
| **Memory** | Separate from compute | Co-located (synaptic weights) | Co-located (synapses) |
| **Forgetting** | Catastrophic | Gradual (bio-plausible) | Gradual consolidation |
| **Hardware fit** | GPUs | Neuromorphic chips (Loihi) | Neurons & dendrites |

## уАЙ Core Problems We Are Solving

### 1 ┬╖ The "Blank Slate" Fallacy
Traditional AI treats intelligence as something to be installed тАФ pour in data, out comes intelligence. But a human infant is not born empty. It is born with reflexes, sensory systems, and a reward/pain engine. Intelligence **emerges** from interaction тАФ it is not injected.

MAYA must be designed as an "infant AGI" тАФ with hardwired drives, not hardwired knowledge.

---

### 2 ┬╖ Catastrophic Forgetting
Every modern neural network is trained once and frozen. If you try to teach it something new, it forgets everything it knew before. This is **Catastrophic Forgetting** тАФ one of the deepest unsolved problems in AI.

MAYA's online learning architecture must solve this fundamentally тАФ not patch it with workarounds.

---

### 3 ┬╖ The Von Neumann Bottleneck
Today's computers store memory and processing in separate physical locations. Moving data between them wastes 90%+ of energy and time. The brain has no such separation тАФ memory and processing occupy the same physical substrate.

MAYA's long-term architecture targets **Neuromorphic Computing** тАФ chips like Intel's Loihi that run Spiking Neural Networks directly, making computation fast, adaptive, and extremely energy-efficient.

---

### 4 ┬╖ Intelligence Without Context
If you feed a system raw data and logic without grounding, it will never understand. Intelligence is not the base. It is a byproduct of surviving in a complex physical world.

MAYA must learn causality, not just correlation. Actions must have consequences.
