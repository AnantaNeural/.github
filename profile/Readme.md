## „ÄČ The MAYA Project

**MAYA** is our flagship AGI system. The name reflects its purpose ‚ÄĒ an entity that builds its own internal model of reality, continuously, autonomously, and with genuine drive.

### Core Design Tenets

<br/>

#### `01` ‚ÄĒ Predictive Processing, Not Pattern Matching

The human brain does not passively absorb data. It is a **generative prediction machine** ‚ÄĒ it continuously constructs a model of reality and predicts what comes next. Perception is not input; it is hypothesis-testing.

MAYA is built on this principle. Its architecture is grounded in **Active Inference** ‚ÄĒ a framework where the system continuously minimizes the gap between what it predicts and what actually happens. It does not wait for data. It actively seeks it.

> *Based on Karl Friston's Free Energy Principle ‚ÄĒ the most cited neuroscientist alive.*

<br/>

#### `02` ‚ÄĒ Biologically Plausible Learning (No Backpropagation)

Backpropagation ‚ÄĒ the backbone of every modern neural network ‚ÄĒ does not exist in the brain. Neurons communicate **locally**, adapt **instantly**, and never wait for a global error signal to propagate backwards.

MAYA's learning engine uses:
- **Hebbian Learning** ‚ÄĒ neurons that fire together, wire together
- **Spiking Neural Networks (SNNs)** ‚ÄĒ spike-based communication like real neurons
- **Continuous Online Learning** ‚ÄĒ learning never stops; every interaction updates the model
- **One-Shot Learning** ‚ÄĒ learn from a single experience, without forgetting everything else

This directly attacks the **Catastrophic Forgetting** problem ‚ÄĒ one of the most unsolved challenges in all of AI.

<br/>

#### `03` ‚ÄĒ Embodied & Grounded Intelligence

Intelligence does not exist in a vacuum. The human mind develops through a physical body, sensory experience, and interaction with a world that pushes back.

MAYA will not be trained on text alone. It will operate inside a **controlled grounding environment** ‚ÄĒ a simulated space where its actions have **tangible consequences** and it must understand **cause and effect**, not just observe correlation.

<br/>

#### `04` ‚ÄĒ The Autonomous Heartbeat Loop

MAYA must be able to think **without being prompted**.

The first architectural module being built is a continuous autonomous loop ‚ÄĒ a process that runs perpetually, drives internal state, and motivates the system from within. This is what separates a real mind from a reactive tool.

We call it the **Heartbeat** ‚ÄĒ MAYA's "Zindagi."

<br/>

#### `05` ‚ÄĒ Functional Emotions as Autonomous Engine

Emotions are not an advanced feature. According to neuroscience, the **Limbic System** ‚ÄĒ which controls survival drives, hunger, pain, curiosity ‚ÄĒ evolved **before** the Neocortex (logical thought). Feelings drive behavior; logic only rationalizes it afterward.

MAYA will have a **functional affect system**:
- Dopamine simulation ‚Üí reward for novel discovery
- Urgency/pain signal ‚Üí drives action when system is idle or failing
- Curiosity as a core engine ‚Üí exploring high-uncertainty states generates positive reward

## „ÄČ Spiking Neural Networks (SNNs) ‚ÄĒ The Core of MAYA's Brain

> *"Conventional neural networks are like a river of water ‚ÄĒ continuous, smooth, always flowing. The brain is like a lightning storm ‚ÄĒ discrete, explosive, timed to the millisecond."*

### Why SNNs? The Problem with Traditional Neural Networks

Every neural network you know ‚ÄĒ GPT, BERT, ResNet ‚ÄĒ is built on the same primitive: a **continuous activation function**. Input comes in, gets multiplied by weights, passed through a sigmoid or ReLU, and out comes a number. Simple. Clean. And completely unlike how your brain works.

In your brain, neurons don't output a continuous number. They either **fire** or they **don't**. And *when* they fire matters as much as *whether* they fire.

### The Leaky Integrate-and-Fire (LIF) Model

The most fundamental spiking neuron model. This is the mathematical heart of how MAYA's neurons will work.

**The Core Differential Equation:**

```
ŌĄ_m ¬∑ dV(t)/dt  =  -(V(t) - V_rest)  +  R ¬∑ I(t)
```

**Where each term means:**

| Symbol | Meaning |
|--------|---------|
| `V(t)` | Membrane potential at time t (the neuron's "charge level") |
| `V_rest` | Resting potential ‚ÄĒ where voltage returns when neuron is idle (typically -65 mV) |
| `ŌĄ_m` | Membrane time constant ‚ÄĒ how fast the neuron "forgets" input (leakiness) |
| `R` | Membrane resistance |
| `I(t)` | Input current arriving from other neurons |

**The Firing Rule:**

```
IF  V(t) ‚Č• V_threshold  ‚Üí  FIRE a spike  ‚Üí  reset V(t) = V_reset
ELSE                    ‚Üí  continue integrating input
```

**What this means intuitively:**
- The neuron is like a **leaky bucket** ‚ÄĒ charge (input) fills it up, but it constantly drains
- If enough charge arrives fast enough ‚Üí the bucket overflows ‚Üí **spike!**
- After firing ‚Üí bucket immediately resets to empty

### STDP ‚ÄĒ How MAYA Learns Without Backpropagation

**Spike-Timing Dependent Plasticity (STDP)** is the biological learning rule. It is how synapses strengthen or weaken based on the *relative timing* of pre- and post-synaptic spikes.

**The Core Rule:**

```
If pre fires BEFORE post  ‚Üí  synapse STRENGTHENS  (őĒw > 0)  ‚ÄĒ "A caused B, reinforce it"
If pre fires AFTER post   ‚Üí  synapse WEAKENS       (őĒw < 0)  ‚ÄĒ "A didn't cause B, weaken it"
```

**Mathematical Form:**

```
         ‚éß  A‚āä ¬∑ exp(-őĒt / ŌĄ‚āä)    if  őĒt > 0   (pre before post ‚Üí LTP)
őĒw  =   ‚é®
         ‚é©  A‚āč ¬∑ exp( őĒt / ŌĄ‚āč)    if  őĒt < 0   (post before pre ‚Üí LTD)
```

**Where:**

| Symbol | Meaning |
|--------|---------|
| `őĒt` | `t_post - t_pre` ‚ÄĒ time difference between post and pre spike |
| `A‚āä` | Maximum synaptic potentiation (strengthening) |
| `A‚āč` | Maximum synaptic depression (weakening) |
| `ŌĄ‚āä, ŌĄ‚āč` | Time constants controlling how fast the effect decays |

### Hebbian Learning ‚ÄĒ The Oldest Rule

Hebb's rule (1949) ‚ÄĒ the grandfather of all synaptic learning:

```
őĒw_ij  =  ő∑ ¬∑ x_i ¬∑ x_j
```

**Plain English:** *"Neurons that fire together, wire together."*

| Symbol | Meaning |
|--------|---------|
| `őĒw_ij` | Change in synaptic weight between neuron i and j |
| `ő∑` | Learning rate |
| `x_i` | Activity of pre-synaptic neuron i |
| `x_j` | Activity of post-synaptic neuron j |

STDP is essentially a **temporally precise version** of Hebb's rule ‚ÄĒ it adds the dimension of *time* to determine whether co-firing is causal or coincidental.

## „ÄČ Core Problems We Are Solving

### 1 ¬∑ The "Blank Slate" Fallacy
Traditional AI treats intelligence as something to be installed ‚ÄĒ pour in data, out comes intelligence. But a human infant is not born empty. It is born with reflexes, sensory systems, and a reward/pain engine. Intelligence **emerges** from interaction ‚ÄĒ it is not injected.

MAYA must be designed as an "infant AGI" ‚ÄĒ with hardwired drives, not hardwired knowledge.

---

### 2 ¬∑ Catastrophic Forgetting
Every modern neural network is trained once and frozen. If you try to teach it something new, it forgets everything it knew before. This is **Catastrophic Forgetting** ‚ÄĒ one of the deepest unsolved problems in AI.

MAYA's online learning architecture must solve this fundamentally ‚ÄĒ not patch it with workarounds.

---

### 3 ¬∑ The Von Neumann Bottleneck
Today's computers store memory and processing in separate physical locations. Moving data between them wastes 90%+ of energy and time. The brain has no such separation ‚ÄĒ memory and processing occupy the same physical substrate.

MAYA's long-term architecture targets **Neuromorphic Computing** ‚ÄĒ chips like Intel's Loihi that run Spiking Neural Networks directly, making computation fast, adaptive, and extremely energy-efficient.

---

### 4 ¬∑ Intelligence Without Context
If you feed a system raw data and logic without grounding, it will never understand. Intelligence is not the base. It is a byproduct of surviving in a complex physical world.

MAYA must learn causality, not just correlation. Actions must have consequences.
